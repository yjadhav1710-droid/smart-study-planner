# 📚 Smart Study Planner

"A Python-based project that helps students organize and manage their daily study schedules effectively."

---

# 📖 Introduction

The Smart Study Planner is a Python project developed to help students manage their study time efficiently. Many students struggle with planning subjects and maintaining consistency during exam preparation.

This project allows users to enter their subjects and daily study hours. The system then generates a simple and organized study plan.

---

# 🎯 Objectives

- To help students create daily study schedules
- To improve time management skills
- To reduce procrastination among students
- To encourage productive study habits
- To build a beginner-friendly Python application

# ⚙️ Methodology

## Step 1:
The program asks the user to enter the number of subjects.

## Step 2:
The user enters subject names and study hours.

## Step 3:
The data is stored using Python lists.

## Step 4:
The system generates and displays the study timetable.

## Step 5:
The total study hours are calculated and analyzed.

# 🚨 Problem Statement

Many students fail to manage their time effectively during exam preparation. Poor planning often leads to stress, procrastination, and low productivity.

This project aims to solve this issue by providing a simple digital study planner that helps students organize their study schedules efficiently.
